Summary

  • SeaSketch is a SaaS platform for hosting spatial planning projects in the Ocean
  • It leverages existing data providers
  • It provides end-users the ability to
    • visualize data
    • develop spatial plans
    • receive analytical feedback
    • and collaborate in a real-time discussion forum

Summary

  • For administrators (planners, managers) it provides a tool to create and administer planning projects, including:
    • inviting users
    • curating a list of data layers
    • creating and moderating discussion forums
    • designing sketch classes to represent planning objectives
    • creating data survey tools
